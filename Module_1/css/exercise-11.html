Debug & Test with DevTools

Testing Guide

Use Chrome DevTools

  1. Open DevTools with F12 or Ctrl+Shift+I
  2. Use the Device Toolbar (Ctrl+Shift+M) to simulate mobile/tablet/desktop sizes
  3. Inspect this card to see applied styles in the Styles panel
  4. Toggle CSS properties on/off to test changes live
  5. Check the Network tab to verify styles.css loaded successfully (status 200)
  6. View the Computed tab to see final box model values
